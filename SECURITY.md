| -------| ------------------ |

| Windows 8 és régebbi | ❌ |

## Használatra alkalmas verziók

| Verzió  | Használhatóság     |
| -------| ------------------ |
| Realase 1.0 | ✅ |


Ahol ❗ található az a verzió nem javasolt, és ahol ❌ van, az használatra alkalmatlan, válassz másik verziót. Ha ⚠️ van akkor az még BETA verzió. Ha pedig ✅ van, az teljesen jó.

## Supported versions

| Version  | Supported |
| -------| ---------|
| Release 1.0 | ✅ |


Where ❗ appears, that version is not recommended; where ❌ appears, it is unsuitable for use—please choose another version. If ⚠️ appears, it is still a BETA version. If ✅ appears, it is fully functional.
